Try to import an article now!

For example: if Internet Explorer is your default browser, press Ctrl+Shift+W, type SuperMemo and press Enter to import an article about SuperMemo from Wikipedia.

If you use another browser, you will need to first open a Wikipedia article in Internet Explorer, and only then press Ctrl+Shift+W.

Remember that you can return to this page by using Alt+Left or the button (at the top of this window).

You can also import articles from other sources. For example:

  1. Open any article in your web browser (for example, right click over this link www.supermemo.com/help/read.htm and choose Open in new window)
  2. Select any portion of text with the mouse (in the browser)
  3. Press Ctrl+C (in the browser)
  4. Return to SuperMemo (e.g. with Alt+Tab)
  5. Press Ctrl+N (in SuperMemo)
